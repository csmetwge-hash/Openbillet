# PortalFlow

Simple and beautiful client portals for solopreneurs, freelancers, and one-person businesses.

## Features
- Magic link client portals (no login required for clients)
- File sharing and storage
- Real-time messaging
- Project status tracking
- Built with Next.js + Supabase + Stripe

## Tech Stack
- Next.js 15
- Supabase (Auth + Database + Storage)
- Stripe Subscriptions
- Tailwind CSS

## Setup

1. Clone the repo
2. Copy `.env.example` to `.env.local`
3. Run `npm install`
4. Run `npm run dev`

## Deploy

Easiest deployment: [Vercel](https://vercel.com)
